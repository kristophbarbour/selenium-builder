# selenium-builder

**How to build**
Go into selenium-buider directory and run:
```bash
jpm xpi --addon-dir seleniumbuilder
```
***For information on jpm xpi [here](https://developer.mozilla.org/en-US/Add-ons/SDK/Tools/jpm)***

Then load the extension in Firefox (e.g. `4wf56ef435w6@jds-0.1.xpi`) by going to about:addons and clicking the cog then __Install Add-on From File__.
